# 概述

该项目是使用 [three.js](https://www.npmjs.com/package/three) 来实现的滚动动画，你可以通过 [这个链接](https://scroll-eta.vercel.app/) 来查看 demo。

![simple](./image-hosting/simple.png)

<br />

# 运行

依次执行下述命令，来执行该项目：

1. `npm install`
2. `npm run start`
